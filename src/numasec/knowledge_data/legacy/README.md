# Legacy CTF Knowledge Base

This directory contains CTF-specific patterns and techniques. These are kept for backward compatibility when NumaSec is used in CTF mode (detected via objective containing 'flag' or 'ctf' keywords).

For **enterprise security assessment**, see the `knowledge/enterprise/` directory.

## Files
- `picoctf_patterns.md` - PicoCTF-specific challenge patterns
- `ctf_formats.md` - CTF flag formats and competition structures
- `ctf_solution_patterns.md` - Common CTF solve patterns

---
**Note:** NumaSec.0+ prioritizes real-world vulnerability detection over CTF flag hunting.
