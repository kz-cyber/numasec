Execute multiple security scanners in sequence on a target.

Parameters:
- session_id (required): Session ID
- target (required): Target URL
- scanners (required): Comma-separated scanner names to run (e.g., "injection_test,xss_test,ssrf_test")
- params: JSON object of shared parameters for all scanners.
- headers: JSON object of additional headers.
- auth_token: Bearer token for authenticated testing.

Returns combined results from all scanners.
