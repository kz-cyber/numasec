Automate browser interactions using Playwright for testing client-side vulnerabilities, capturing screenshots, and interacting with JavaScript-heavy applications.

Parameters:
- target (required): Target URL to open
- actions: JSON array of browser actions to perform. Each action is {"type": "click|type|wait|screenshot|evaluate", ...}.
- wait_for: CSS selector or time (ms) to wait before capturing. Default: "networkidle".
- screenshot: Whether to capture a screenshot. Default: false.
- headers: JSON object of additional headers.
- auth_token: Bearer token for authenticated browsing.

Returns browser interaction results, page content, and optionally screenshots.
