Send a custom HTTP request for manual testing and analysis.

Parameters:
- url (required): Full URL to send the request to
- method: HTTP method — GET, POST, PUT, DELETE, PATCH, OPTIONS, HEAD. Default: "GET".
- headers: JSON object of request headers.
- data: Request body (string or JSON object).
- params: JSON object of query parameters.
- auth_token: Bearer token to include.
- follow_redirects: Whether to follow redirects. Default: true.
- timeout: Request timeout in seconds. Default: 30.
- verify_ssl: Whether to verify SSL certificates. Default: true.

Returns the full HTTP response including status code, headers, body, and timing.
