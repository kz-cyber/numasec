Save a security finding with automatic enrichment (CWE, CVSS, OWASP category, MITRE ATT&CK).

Parameters:
- session_id (required): Session ID from create_session
- title (required): Finding title (e.g., "SQL Injection in login endpoint")
- severity (required): Severity — "critical", "high", "medium", "low", "info"
- description (required): Detailed description of the vulnerability
- url: Affected URL
- method: HTTP method used
- parameter: Vulnerable parameter name
- evidence: Evidence of the vulnerability (payload, response, etc.)
- cwe_id: CWE identifier (e.g., "CWE-89"). Auto-detected if not provided.
- remediation: Recommended fix

Returns the saved finding with enriched metadata including CVSS score and OWASP category.
