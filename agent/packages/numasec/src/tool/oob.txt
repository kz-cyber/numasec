Test for out-of-band (OOB) interaction vulnerabilities using external callback servers.

Parameters:
- target (required): Target URL to test
- method: HTTP method. Default: "GET".
- params: JSON object of parameters to inject OOB payloads into.
- data: Request body for POST requests.
- headers: JSON object of additional headers.
- callback_url: External OOB callback URL. If not provided, uses built-in listener.
- payload_types: Comma-separated types. Options: dns, http, smtp. Default: "dns,http".

Returns detected OOB interactions with timing and source information.
