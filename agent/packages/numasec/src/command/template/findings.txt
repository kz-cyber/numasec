List all findings from the current pentest session.

Command migration note:
- Canonical command: `/finding list`
- Legacy alias: `/findings` (soft-deprecated, still supported in v1.x)

Call `get_findings` to retrieve all findings, then display them in a table sorted by severity (Critical > High > Medium > Low > Info).

For each finding show:
- Severity (with color indicator)
- Title
- URL / endpoint
- OWASP category
- CWE ID
- Confirmed status

At the end, show a summary: total findings by severity, and OWASP Top 10 coverage percentage.
