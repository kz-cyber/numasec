Determine the next verification step for the current session.

1. Call `query_graph` with:
   - node_types=["hypothesis","observation","artifact","verification","finding"]
   - include_edges=true
   - limit=200
2. Infer current planner context:
   - active hypothesis (open/probing, highest confidence)
   - available evidence node IDs
   - verification progress (supports/refutes/passed)
3. Call `plan_next` with the inferred context.
4. Return:
   - planner state
   - next primitive
   - exact next action to run (tool + key parameters)
   - missing evidence required to confirm or reject
