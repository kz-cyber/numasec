Show the current engagement scope and latest observed target surface.

Call `query_graph` with:
- node_types=["artifact"]
- include_edges=false
- limit=200

From the returned nodes, find the newest artifact produced by `observe_surface` (source_tool="observe_surface") and summarize:
- Target
- Observation modes
- Open ports/services
- Technologies
- Endpoint/form/secret counts
- Last updated timestamp

If no scope/surface artifact is found, tell the user no scope is currently defined and suggest:
- `/scope set <target>` (preferred)
- `/target <target>` (legacy alias, soft-deprecated in v1.x)
